#  Flutter News Application (API Integration)

A modern Flutter application that fetches real-time news data from a free online API and displays news articles in a clean and structured list format.

Each news item includes:
-  Image
-  Title
-  Date
-  Description

---

##  Features

-  Fetch live news from API
-  Display news articles in ListView
-  Show article images
-  Title, date, and description included
-  Clean and scrollable UI
-  Dynamic data loading from API

---
